Classification Settings

RMSE  √(Σ(|Z|sim−|Z|exp)² / N) — penalizes large absolute deviations. Sensitive to outliers.
Log-RMSE  √(Σ(log|Z|sim−log|Z|exp)² / N) — scale-invariant; balances contributions across all frequencies. Recommended.
MAE  Σ||Z|sim−|Z|exp| / N — linear penalty, more robust to outliers than RMSE.
Log-MAE  Σ|log|Z|sim−log|Z|exp|| / N — robust log-scale comparison.
Relative Error  Σ||Z|sim−|Z|exp| / |Z|exp / N — normalized to experimental magnitude; useful when |Z| spans many orders of magnitude.
All metrics compare interpolated simulation |Z| vs experimental |Z| (with Rconstr subtracted from exp). Lower = better fit.
